### 食用指北:

1. 先将此库下载到本地

前置问题:Windows设置git、python等环境变量

Linux & Mac: 包管理: Homebrew(Mac) , apt/rpm...(Linux)  $\to$下载git,python 

- GitHub:
```bash
git clone https://github.com/ACpeel/HEn.git
```

- Gitee:

```bash
git clone https://gitee.com/chenyh43/HEn.git
```

2. 下载相应的依赖:Pygame

```bash
cd HEn/src
pip install -r requrements.txt
```

等待下载完成



3. 启动 !

```bash
python3 main.py
```
